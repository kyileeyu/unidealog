# Welcome to Unidealog! 🎉

안녕하세요! **Unidealog**에 오신 것을 환영합니다. 이것은 Next.js 13+ App Router와 shadcn/ui로 구축된 현대적인 블로그 플랫폼입니다.

## 🚀 주요 특징

### 현대적인 기술 스택
- **Next.js 13+**: App Router를 사용한 최신 React 프레임워크
- **TypeScript**: 타입 안전성을 보장하는 정적 타입 언어
- **Tailwind CSS**: 유틸리티 퍼스트 CSS 프레임워크
- **shadcn/ui**: 아름답고 접근 가능한 UI 컴포넌트

### Feature-Sliced Design (FSD)
이 블로그는 FSD 아키텍처를 따라 구조화되어 있습니다:

```
src/
├── shared/     # 공유 자원 (UI 컴포넌트, 유틸리티)
├── entities/   # 비즈니스 엔티티 (Post, Category, User)
├── features/   # 사용자 기능 (Search, Theme, Navigation)
├── widgets/    # 복합 UI 블록 (Header, Footer, PostCard)
└── pages/      # 페이지 컴포넌트
```

## ✨ 블로그 기능

### 📝 MDX 지원
마크다운과 JSX를 함께 사용할 수 있어 풍부한 콘텐츠를 작성할 수 있습니다.

### 🔍 검색 기능
포스트 제목, 내용, 태그를 통한 실시간 검색이 가능합니다.

### 🏷️ 태그 및 카테고리
콘텐츠를 체계적으로 분류하고 필터링할 수 있습니다.

### 🌓 다크 모드
사용자 취향에 맞는 테마를 선택할 수 있습니다.

## 🎯 앞으로의 계획

이 블로그는 지속적으로 발전할 예정입니다:

1. **콘텐츠 확장**: 다양한 주제의 포스트 추가
2. **기능 개선**: 댓글 시스템, 소셜 공유 등
3. **성능 최적화**: 이미지 최적화, 캐싱 전략
4. **SEO 강화**: 검색 엔진 최적화
